# MFormula · Member Day — 解锁好礼 dashboard (可点击原型)

老顾客专属活动(1/7–7/7)。核心:**下单不被游戏挡住** —— 想直接下单就选配套找客服;想要免费好礼,就玩**几秒钟的敲击小游戏**解锁好礼,搭配到配套里,再一键 WhatsApp 找客服下单。
纯前端 demo,真实上线需 Backend(防作弊/记账/兑换核销),见最下方。

## 怎么打开
- 双击 `index.html`(手机视角最佳,F12 切手机模拟)
- 或:`node server.js` → http://localhost:4599

## 玩法(为"不拖时间"设计)
1. **一打开就是两个配套**:2 Boxes RM358 / 4 Boxes RM716。想直接下单 → 选配套 → 联系客服。**不用玩游戏。**
2. **下面是一排礼物 + 折扣券**,都锁着。点一个 → 玩 **5 秒敲击充能**,把能量条敲满就**解锁**。
3. **配套决定能搭几个**:2 Boxes 搭 **2 个**,4 Boxes 搭 **3 个** → 把人往 716 推。
4. **联系客服 WhatsApp**:把配套 + 搭配的好礼 + 兑换码自动填好发给客服下单。

## 敲击小游戏(关键)
- 5 秒内狂敲,把能量条敲满目标下数就解锁。
- **手速上限**:两次有效敲击至少隔 `90ms` → 5 秒最多约 **55 下**。
- 普通奖励目标 12~47 下(够得着);**免单 / 黄金大奖目标 62 / 66 下 —— 物理上敲不满**,但能量条会冲到 ~85% 卡在终点前,制造"就差一点!再来一次"。连脚本都刷不过,玩家却觉得能赢。

## 奖励清单(用你的真实礼物 · 都可改)
| 难度 | 奖励 |
|---|---|
| 易 | 1 包 / 5 包 Sachet · 折扣 RM10 |
| 中 | FIFA Tumbler · 10 包 Sachet · 折扣 RM20 |
| 难 | FIFA Duffle Bag · 20 包 Sachet · 1 盒 MFormula · 折扣 RM30 |

> 免单/黄金 = 独占,选了就不能再叠别的;普通折扣之间可叠加。

## ⚙️ 你要改的地方(都在 `app.js` 顶部)
- **`CONFIG.WHATSAPP`** ← 你的客服 WhatsApp 号(国际格式不带+,如 `60123456789`)。**现在是占位号!**
- `CONFIG.DAILY_CHANCES` 每天敲击机会 · `TAP_WINDOW` 秒数 · `MIN_TAP_GAP` 手速上限(调难度)
- `REWARDS` / `ULTRA` 奖励清单 + 每个的 `target`(敲几下解锁,越大越难)
- `PACKAGES` 配套价 + `slots`(能搭几个)
